Control System

PID Algorithm &
Active Stabilization

Arduino Uno R3 (ATmega328P) reads DHT22 internal climate sensor and YL-69 soil reference. Ceramic heater + dual fan array responds in real-time via PID loop to maintain artifact-safe environment.

// arduino.ino — Control Logic
MCU: Arduino Uno R3 (ATmega328P)
Sensors: DHT22, YL-69
Actuators: Ceramic Heater, 2x Fans
Algorithm: PID (Kp=2.0, Ki=0.5, Kd=1.0)
